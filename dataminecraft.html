Below are some of the interactive data visualizations I've created recently. Most of them are written in R, D3 and JavaScript, while I explored Gephi, Wolfram Alpha, Many Eyes, Shiny etc. at the same time. I named this section "Data Minecraft" because of this post I saw and found interesting the other day -- contrary to common impression of being boring, I think data can be an eye candy too! I've attached link to code in each project and they can be found in my Github all together.

December 2012 Top Twitter Keywords Revealing Major News

Rachel Shuyan Wang

This is an interactive Streamgraph in D3.js I've created using the top 10 keywords in Twitter December 2012 extracted from the Tweet data sample. I have always been curious about how soon society will react to a breaking news on social media and how long it takes for the attention to "fade away", as my guess is that people are becoming more forgetful in the Digital Age. By moving your mouse onto highlighted part, you can see the story behind the keyword and the frequency it is appearing on Twitter.

Is Net Approach the Key to Winning

Rachel Shuyan Wang

This is an interactive and comparable Treemap I've created using the top 30 male players in US Open from 2003 to 2013. In Tennis, it’s certainly true that a skilled player could win more points by moving forward. But when pros don’t emphasize that part of their game and they gain little match experience approaching the net, if they have the skills necessary to take advantage of such an opportunity is a question to be answered.
