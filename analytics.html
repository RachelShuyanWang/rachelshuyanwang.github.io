Predicting Yelp Review Upvotes by Mining Underlying Topics

Rachel Shuyan Wang

When a restaurant has already have more than a hundred reviews on Yelp (or Youtube, Airbnb, etc), are you still willing to write a new one knowing that it might be buried deep? The lack of user engagement would reduce Yelp traffic, and slow down the metabolism of the website in the long run. This paper applied LDA and other topic modeling methods on the Yelp Academic Dataset to design a system that can automatically recommend potentially high quality reviews. It also discovers the positive correlation between the amount of 'useful' subtopics a review covers and its usefulness decided by the human annotators.

All code for this research project is written in R, including data cleaning, text parsing (text mining package 'tm'), statistical model ('lda' and 'topicmodels') and data visualization ('ggplot2', 'wordcloud', scatter plot, heatmap etc.).
